# Nodepop

## API for a 2nd hand articles App developed in NodeJS + Express4 + Mongoose

### Practice for KeepCoding Mobile Bootcamp 8 - Node Fundamentals

## Installation Instructions



Run MongoDB

```shell
bin/mongod --dbpath ./data/db --directoryperdb
```
Start database
```shell
npm run installDB
``` 

Start app
```shell
npm run dev
```
